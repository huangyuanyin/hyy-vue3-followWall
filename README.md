### 项目启动

```
pnpm install
pnpm run dev
```

### 创建项目

```
pnpm create hyy-vue3-followWall
```

### 安装路由插件

```
pnpm install router --save
```

### 安装 vuex 插件

```
pnpm install vuex --save
```

### 安装 less 插件

```
pnpm install less less-loader --save
```

### 安装 axios 插件

```
pnpm install axios --save
```

如果在 vue3 中直接使用，安装 vue 扩展 axios 插件

```
pnpm install vue-axios --save
```
